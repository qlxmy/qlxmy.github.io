apk反编译

发表于 2020-01-14 分类于 移动安全

dex2jar获得jar代码文件

dex2jar.bat apk文件

也可以直接将apk文件拖放到dex2jar.bat上。
或者使用第二种办法，将apk当成zip解压，得到如下文件：

用dex2jar.bat将classes.dex进行转换：
dex2jar.bat classes.dex

jd-gui读取jar文件

使用jd-gui来读取jar文件：

选择File-Save All Sources保存成zip压缩包，便可将其还原成java文件。

apktool调试apk文件

解包apk文件：
java –jar apktool.jar d –f apk 源文件路径 –o 目标文件路径

重新打包apk文件：
java –jar apktool.jar b –f 源目录 –o 目标apk文件路径

解包后的apk可以直接查看smali文件：

也可以直接读取AndroidManifest.xml的内容：

keytool生成证书文件

keytool生成证书文件：
keytool -genkey -v -keystore my_key.Keystore -alias android_key -keyalg RSA -keysize 2048 -validity 9999

jarsigner对apk文件签名

jarsigner -verbose -sigalg SHA1withRSA -digestalg SHA1 -keystore my_key.Keystore test.apk android_key

signapk对apk文件签名

signapk.jar对apk文件进行签名：
java –jar signapk.jar testkey.x509.pem testkey.pk8 源apk路径 目标apk路径

Keytool查看签名

使用java的keytool工具可以查看apk签名，签名文件主要保存在META-INF目录下：
keytool –printcert –file META-INF/CERT.RSA

jarsigner验证签名

jarsigner -verify -verbose -certs test.apk

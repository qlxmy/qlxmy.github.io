Ubuntu安装JDK和Tomcat

发表于 2020-01-14 分类于 Linux系统

安装准备

安装环境如下：
Ubuntu版本：Ubuntu 16.04.3
jdk版本：jdk-8u144-linux-x64
tomcat版本：apache-tomcat-8.5.23
首先下载jdk和tomcat，下载页面如下：
JDK: http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
Tomcat：https://tomcat.apache.org/download-80.cgi

安装JDK

解压jdk压缩包：

tar xzvf jdk-8u144-linux-x64.tar.gz

创建安装目录并将解压的jdk文件夹移动到该目录：

sudo mkdir /usr/lib/jvm
sudo mv jdk1.8.0_144/ /usr/lib/jvm

配置/etc/profile文件，在后面添加以下内容：

export JAVA_HOME=/usr/lib/jvm/jdk1.8.0_144  #修改为自己的安装目录
export JRE_HOME=${JAVA_HOME}/jre  
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib  
export PATH=${JAVA_HOME}/bin:$PATH

然后让配置生效：

source /etc/profile

最后查看jdk版本：

Java –version

安装Tomcat

解压Tomcat压缩包：

tar xzvf apache-tomcat-8.5.23.tar.gz

将解压出来的文件夹移动到要安装的目录中：

sudo mv apache-tomcat-8.5.23/ /opt

配置tomcat目录下的bin/startup.sh文件：

sudo vim ./bin/startup.sh

添加如下内容：

JAVA_HOME=/usr/lib/jvm/jdk1.8.0_144    #修改为自己jdk的安装目录
JRE_HOME=$JAVA_HOME/jre
PATH=$PATH:$JAVA_HOME/bin:$JRE_HOME
CLASSPATH=.:$JRE_HOME/lib/rt.jar:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
TOMCAT_HOME=/opt/apache-tomcat-8.5.23        #修改为自己Tomcat的安装目录


然后启动tomcat：

sudo ./bin/startup.sh

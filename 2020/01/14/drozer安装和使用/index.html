drozer安装和使用

发表于 2020-01-14 分类于 移动安全

Drozer是一款非常不错的Android APP安全评估工具，下面介绍如何安装和使用它。

Drozer下载

Drozer的下载地址是https://labs.mwrinfosecurity.com/tools/drozer/，需要下载PC端和手机agent.apk，我下载的是Windows版本：

Drozer安装

在PC上安装Drozer：

在手机或者Android模拟器上安装Drozer，我这里在夜神Android模拟器上安装：

Drozer连接

在Windows命令提示符窗口中输入“Drozer”查看使用方法：

console 运行Drozer命令行窗口
module　　　管理Drozer组件
server 　　　运行Drozer服务器
ssl　　　　　　管理Drozer SSL密钥
exploit 　生成可以在Drozer中使用的攻击代码
agent 　生成用户定义的Drozer agent
payload 生成可以在Drozer中使用的Payload

在模拟器中运行Drozer：

可以看到端口为31415，单击“Embedded Server”，可以看到运行状态：

首先PC端要连接Android模拟器：

62001是夜神模拟器的端口，其他的模拟器请替换成相应的端口。
接着连接Drozer端口：

adb forward tcp:31415 tcp:31415


最后运行Drozer命令行窗口：

drozer console connect


出现如上界面说明运行成功了。

Drozer使用

Drozer的命令

输入list命令可以列出所有能在Drozer中使用的命令：

下面对这些命令做个简单的介绍：

app.activity.forintent 查询可处理Intent的Activity
app.activity.info 获取可导出的Activity信息
app.activity.start 启动Activity
app.broadcast.info 获取BroadcastReceiver信息
app.broadcast.send 向正在使用的Intent发送广播
app.broadcast.sniff 注册一个可以监听特定意图的广播接收器
app.package.attacksurface 获取Package的攻击面
app.package.backup 查看Package是否允许备份功能
app.package.debuggable 查找可调试的Package
app.package.info 获取Package信息
app.package.launchintent 访问Package的Intent
app.package.list 列出设备上已安装的Package
app.package.manifest 获取Package的AndroidManifest.xml
app.package.native 查找嵌入在应用程序中的本地库
app.package.shareduid 查找Package的共享UID
app.provider.columns 查看内容提供器列表
app.provider.delete 删除内容提供器
app.provider.download 从支持文件的内容提供器处下载文件
app.provider.finduri 查询Package中浏览的内容URI
app.provider.info 获取可导出的内容提供器信息
app.provider.insert 插入内容提供器
app.provider.query 查询内容提供器
app.provider.read 读取支持文件的内容提供器
app.provider.update 升级内容提供器中的记录
app.service.info 查看可导出的服务
app.service.send 给服务发送信息，并显示响应结果
app.service.start 启用服务
app.service.stop 停止服务
auxiliary.webcontentresolver 启用内容提供器的Web服务界面
exploit.jdwp.check 打开@jdwp-control查看app连接情况
exploit.pilfer.general.apnprovider 读取APN内容提供器
exploit.pilfer.general.settingsprovider 读取内容提供器设置
information.datetime 显示日期和时间信息
information.deviceinfo 显示设备详细信息
information.permissions 显示所有Package的权限
scanner.activity.browsable 获取所有可以从Web浏览器调用的组件
scanner.misc.native 扫描Package中的本地组件
scanner.misc.readablefiles 扫描指定文件夹下的可读文件
scanner.misc.secretcodes 扫描可用于拨号的安全代码
scanner.misc.sflagbinaries 扫描系统目录下的suid/sgid二进制文件
scanner.misc.writablefiles 扫描指定文件夹下的可写文件
scanner.provider.finduris 扫描可查询的内容提供器
scanner.provider.injection 测试存在SQL注入漏洞的内容提供器
scanner.provider.sqltables 扫描可通过SQL注入漏洞访问的表
sannner.provier.traversal 测试存在目录遍历漏洞的内容提供器
shell.exec 执行单条Linux shell
shell.send 向远程监听者发送ASH shell
shell.start 输入交互式的Linux shell
tools.file.download 下载文件
tools.file.md5sum 获取文件的md5校验和
tools.file.size 获取文件大小
tools.file.upload 上传文件
tools.setup.busybox 安装Busybox
tools.setup.minimalsu 准备在设备中安装minimal-su程序

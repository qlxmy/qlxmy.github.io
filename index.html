安装准备

安装环境如下：
Ubuntu版本：Ubuntu 16.04.3
jdk版本：jdk-8u144-linux-x64
tomcat版本：apache-tomcat-8.5.23
首先下载jdk和tomcat，下载页面如下：
JDK: http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
Tomcat：https://tomcat.apache.org/download-80.cgi

安装JDK

解压jdk压缩包：

tar xzvf jdk-8u144-linux-x64.tar.gz

创建安装目录并将解压的jdk文件夹移动到该目录：

sudo mkdir /usr/lib/jvm
sudo mv jdk1.8.0_144/ /usr/lib/jvm

配置/etc/profile文件，在后面添加以下内容：

export JAVA_HOME=/usr/lib/jvm/jdk1.8.0_144  #修改为自己的安装目录
export JRE_HOME=${JAVA_HOME}/jre  
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib  
export PATH=${JAVA_HOME}/bin:$PATH

然后让配置生效：

<?php  
//heredoc  
$title = "heredoc的用法";  
$str1 = <<<heredoc  
<h1>$title</h1>  
<p>当需要写一大段文字的时候，这段文件包含了单引号，双引号，还有转义符，用其他方式来输出往往  
比较麻烦。\n这时候可以用heredoc。</p>  
<p>heredoc的语法如下：</p>  
1.开始和结尾需要插入标识符；<br />  
2.标识符和名称和变量名的要求一样；<br />

2.nowdoc相当于加长版的单引号标识，不会解析变量和转义序列。<br />

This is a test.\n$title  
nowdoc;

include和require

发表于 2020-01-14 分类于 PHP学习

PHP包含文件最常用的两种方式是include()和require()。

include()和require()

include()的语法如下：

include(/path/to/filename)

和print、echo一样，include()可以省略括号，使用引号来代替。include包含的代码必须使用PHP界定符包含起来，否则是不能正确解析的。如果启用了PHP配置指令allow_url_fopen，include()还可以包含一个远程文件。

require()和include()相似，但它们有以下几个区别：
1.include()有返回值，require()没有。
2.require()是无条件包含，即将require()放入条件判断语句中，无论判断真假require()都会执行。
3.require()出错时，脚本将停止执行；而include()出错时，脚本会继续往下执行。

include_once()和require_once()

和include()、require类似，include_once()和require_once()同样用于包含文件，但它们会在包含文件前首先确认该文件是否已经被包含了，如果已经包含了文件，则include_once()和require_once()就不再执行。
